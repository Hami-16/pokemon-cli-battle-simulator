# pokemon-cli-battle-simulator
Pokemon CLI Battle Simulator

A command-line Pokémon battle simulator built in Python.

The program allows players to choose a starter Pokémon
(Froakie, Fennekin, or Chespin) and participate in a
turn-based battle against a rival trainer.

Features
- Starter Pokémon selection
- Turn-based battle system
- Randomized move damage
- Pokémon data using object-oriented design
- Pokédex information system

Technologies
Python
Object-Oriented Programming

Purpose
This project was built as a learning exercise to practice
Python programming, OOP concepts, and game logic design.

Future Improvements
- More Pokémon
- Status effects
- Type advantages
- Multiple battles
